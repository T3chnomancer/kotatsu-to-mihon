# kotatsu-to-mihon
step 1: back up your create a data backup on kotatsu \n
step 2: unzip .bk.zip file and rename history to history.json \n
step 3: run the script in the same folder as history.json file \n
step 4: open converted_main.json in https://backup.mihon.tools/ and click at the top right and download as .tachibk file \n
step 5: restore back up in mihon
