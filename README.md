# kotatsu-to-mihon
step 1: back up your create a data backup on kotatsu
step 2: unzip .bk.zip file and rename history to history.json
step 3: run the script in the same folder as history.json file
step 4: open converted_main.json in https://backup.mihon.tools/ and click at the top right and download as .tachibk file
step 5: restore back up in mihon
